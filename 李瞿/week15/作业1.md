## 总结

三个方案都依赖于多种类型的外部数据源：

1. **金融数据**：主要通过 AkShare 库获取，这是最核心的数据来源
2. **网络搜索**：普遍使用 DuckDuckGo 和搜狗搜索进行信息检索
3. **网页解析**：均采用 BeautifulSoup 进行网页内容解析
4. **动态抓取**：部分方案使用 Selenium 进行动态网页内容抓取
5. **大模型服务**：个别方案接入 DashScope 或 OpenAI 的大模型 API
6. **HTTP 请求**：所有方案都使用 requests 库进行 HTTP 通信
7. **专业数据源网站**：第三种方案特别整合了多个专业金融数据源网站
