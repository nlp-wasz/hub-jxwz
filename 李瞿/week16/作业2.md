# 检查是否已经存在于列表中，避免重复
```commandline
if not self.redis.lrem(self.routes_key, 0, question_hash):
    # 将问题哈希值添加到路由列表中
    pipe.lpush(self.routes_key, question_hash)
```
lrem(key, 0, value) 的作用是：删除列表中所有等于 value 的元素（因为 count=0 表示删除全部匹配项）。
若不存在key时，则正常添加key。
如果 question_hash 存在 → 删除 1 个（或多个）→ 返回 ≥1 → not (≥1) 为 False → 不执行 lpush ❌，但把已存在的 hash 给删掉了！
