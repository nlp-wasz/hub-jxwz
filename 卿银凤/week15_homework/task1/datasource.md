# 08-financial-report-generator 方案 - 外部数据源总结

**使用的数据源：**
- 火山引擎 Web API（推理模型：deepseek-v3-250324）

## 二、方案-天池三轮车

**数据获取方式：** 多源数据采集 + RAG + 搜索引擎

**使用的数据源：**

### 官方/权威数据
- **巨潮资讯**：爬取上市公司近五年财报链接，存储为Excel后进行Embedding存入ES库

### 财务数据API
- **东方财富**：获取上市公司财务信息（市盈率、市净率、市销率等）
- **AKShare**：获取上市公司财务数据
- **新浪财经**：获取上市公司财务信息

### 搜索引擎
- **百度搜索**
- **必应搜索（Bing）**
- **搜狗搜索**
- **DuckDuckGo搜索**
- **Google搜索**

### 模型服务
- **阿里云百炼平台**：deepseek-r1-0528、qwen-vl-max
- **ModelScope**：bge-large-zh（Embedding模型）、bge-reranker-base（Rerank模型）
- **MinerU2.0**：PDF抽取模型

---

## 三、方案-队伍名字不能为空

**数据获取方式：** 基于数据字典的RAG（Data Menu RAG）+ MCP服务

**使用的数据源：**

### 官方/权威数据
- **国家统计局**：宏观经济数据
- **中国人民银行统计司**：金融统计数据
- **国务院政策文件库**：政策文件
- **政府网**：政策信息

### 交易所数据
- **上海证券交易所（上交所）**：公司公告、财报
- **深圳证券交易所（深交所）**：公司公告、财报
- **香港交易所（港交所）**：公司公告、财报
- **恒生指数**：指数数据

### 财经数据平台
- **AKShare**：股票数据、外汇数据、期货数据、宏观经济数据
- **百度股市通**：股票K线、板块数据、财务摘要、三大财务报告
- **雪球**：股票数据

### 新闻资讯
- **中国新闻网**：财经新闻

### 模型与中间件服务
- **阿里云百炼平台**：qwen3-235b-a22b 大模型
- **Zilliz Cloud（阿里云托管）**：Milvus向量数据库
